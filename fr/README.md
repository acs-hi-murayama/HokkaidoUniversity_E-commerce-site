# fr/ — お知らせ / フリーページ

らくうるカートの「お知らせ設定（フリーページ）」に相当するフォルダ。
本番URL `/fr/{freepageSerialNo}` に対応させ、ローカルでは `fr/{番号}.html` として格納する。

## 新しいお知らせの作り方

1. `_template.html` をコピーして `fr/` 直下に置く（例：`fr/4.html`）。
2. ファイル内の以下を書き換える。
   - `__PAGE_TITLE__` … お知らせタイトル
   - `__PAGE_DATE__` … 日付（例：2026.07.13）
   - `__PAGE_BODY__` … 本文HTML
3. トップページ [../index.html](../index.html) の「お知らせ（NEWS）」リストに
   `<dd><a href="./fr/4.html">タイトル</a></dd>` を追加する。
   （本番ではカートが `freepageList` から自動生成するため、この手動追加は不要になる）

## 本番移行時のメモ

- 本文（`freepage-body` の中身）だけをカートの「お知らせ本文」に貼り付ける。
- ヘッダ/フッタはカート側の `Common_Part.html` が付与するので、移行時は不要（本文のみ）。
- 詳細は [MAPPING.md](../docs/MAPPING.md) を参照。
