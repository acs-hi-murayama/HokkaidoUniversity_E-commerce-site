# 実素材 収集リスト（本番移行用）

ローカルのダミーデータを実データに差し替え、らくうるカートへ移行するために集める素材の一覧。
詳細な変数対応は [MAPPING.md](MAPPING.md) を参照。

---

## 必須（これが無いと移行できない）

### ① 商品データ（各商品ごと）　✅ 反映済み（2026-07-13）
らくうるカートの商品エクスポートCSVを受領し、`data/products.json`（実175件）に整形。
トップ/一覧/検索/詳細の4ページを実データ・実画像URLで描画するよう対応済み（`js/shop.js`）。
※カテゴリ名マスタは未受領のため、サイドバーのカテゴリ名のみ引き続きダミー。

1商品につき以下がワンセット。

| 項目 | Twig変数 | 備考 |
|---|---|---|
| 商品名（正式名称） | `item.itemName` / `itemData.itemName` | `item6` のゆらぎを確定（index=パーカー／list=エコバッグ） |
| 商品画像URL（サムネ） | `item.dispMediaUrlSmall` / `mediaUrlSmall` | カート登録済みなら自動発行。**手集めは基本不要**（ローカル確認用のみ） |
| 価格（税込表示） | `item.displayPrice` / `displayTax` | 例：¥1,650（税込） |
| 商品コード / JAN | `variationList[0].itemCd` / `jancode` | あれば |
| カテゴリ / バリエーション | `itemData.categoryName` / `variation.variationName` | サイズ・色など（例：パーカーMサイズ） |

> 商品画像は本番ではカートが `{{ item.dispMediaUrlSmall }}` で自動展開するため、**実URLを手集めする必要は基本ない**。ローカルで見た目を実物に近づけたい場合のみ収集。

## LP関連（本番流用を見据えるなら重要）

### ④ LP素材
| 項目 | 備考 | 状態 |
|---|---|---|
| LP本文の最終テキスト | 見出し・説明文・キャッチコピー | |
| LP内画像URL | 現在 `lp1〜3`（400×300）。**実URL直書きになるため実URLが必要** | |
| 既存LP素材の扱い | `original/HTML/Sanrio_Clearfile.html` を流用/踏襲するか判断 | |

> LP本文はカートの「自由記述（WYSIWYG）」= `MAIN_CONTENTS` に入る。ローカルで作り込んだ本文をそのまま移植できる（最も再利用しやすい）。

## 優先順位まとめ

1. **商品名・価格の確定**（①テキスト部分）← 今すぐ着手可・`item6`問題も解消
2. **LP本文＋LP画像URL**（④）← 本番流用の価値が高い
3. **ロゴ・キービジュアル・ファビコン**（②③）
4. **共通情報・お知らせ**（⑤）

> ポイント：**商品画像URLの手集めは基本不要**（カートが自動展開する）。手を動かすべきはテキスト確定とLP素材。
